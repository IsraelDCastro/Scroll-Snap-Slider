# Scroll-Snap-Slider

Un simple Slider usando la propiedad `scroll-snap` de CSS.

El JS se usa para cambiar la posición de los elementos tomando el cuenta su tamaño con la propiedad `scrollWidth`, de esta manera moverlos de lugar.
